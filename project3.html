This Website

As my ability grows, so does my portfolio website. One could say we evolve together. I am constantly adding new elements, redesigning its components, trying new color schemes, tinkering with animations, etc. From its first iteration to what you are seeing now, it truly represents my journey as a software developer.

I wanted to provide my visitors with a user interface built with a clean neon design. Utilizing a carefully selected color scheme combined with some CSS magic, I gave select elements a neon glow effect. I also created my logo, which one can find serving different purposes throughout the site, using Adobe Illustrator.
